## Install
```
git clone https://github.com/yahyamourid/healthTime

```
### Using Docker 
```
docker-compose build
docker-compose up
```
- Connect to the PhpMyAdmin server in Docker on port 8081 with the credentials 'root' for both username and password
- Restart the back-end project from docker desktop





### Using mvn and npm
- cd healthtime-backend
```
mvn clean install
mvn spring-boot:run 
```

- cd healthtime-frontend
```
npm install
npm run dev
```
- visualiser le site-web en local
```
 http://localhost:5173/
```
### Data base
- Import the 'healthTime.sql' database file into your local MySQL server
